# TableFlow

TableFlow is a QR Code restaurant ordering system built with Django.

Customers scan a QR code at their table, open the menu, and place orders directly from their phone. Orders are instantly sent to the kitchen dashboard.

## Features

- QR Code per table
- Digital menu
- Customer ordering from phone
- Kitchen dashboard
- Order status management

## Tech Stack

- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite

## How it Works

1. Each table has a QR code
2. Customer scans the QR code
3. Menu opens on their phone
4. Customer places order
5. Order appears in the kitchen dashboard
